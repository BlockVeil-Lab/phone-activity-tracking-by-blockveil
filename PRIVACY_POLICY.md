# Privacy Policy

phone-activity-tracking-by-blockveil respects user privacy by design.

## Data Collection
This project does **not** collect, store, transmit, or process any personal data on behalf of BlockVeil or any third party.

All activity data (such as screen events or connectivity status) is:
- Generated locally on the user’s Android device via MacroDroid
- Sent directly to the user’s own Google Apps Script endpoint
- Stored only in the user’s own Google Sheets account

## Data Ownership
- All data belongs 100% to the user
- BlockVeil has no access to user data
- No shared servers, databases, or analytics are used

## Third-Party Services
This project relies only on:
- MacroDroid (local automation on the device)
- *Google Sheets & Google Apps Script (user-controlled Google services)

BlockVeil does not operate or control these services.

## Network Usage
- Data is transmitted only via HTTP POST from the user’s device to their own Apps Script URL
- No background tracking or hidden network activity exists

## Security
- No authentication tokens or credentials are collected
- No logs are stored outside the user’s Google account

## Changes
This privacy policy may be updated if the project scope changes. Any changes will be documented publicly in this repository.
